=================================
Open-Source & Third-Party Tools
=================================

The Only Menu project relies on several open-source and third-party tools to deliver a scalable and efficient solution.

**Frontend**

- **React.js/Vue.js**: Framework for building the PWA interface.
- **React Dropzone**: Used for handling file uploads.

**Backend**

- **FastAPI**: High-performance API framework for Python.
- **Flask**: Lightweight WSGI web application framework for Python.
- **Express.js**: Web application framework for Node.js.

**AI & Image Processing**

- **TensorFlow**: Open-source library for machine learning.
- **Tesseract**: OCR engine for extracting text from images.
- **PyTorch**: Framework for building and training AI models.

**Database**

- **PostgreSQL**: Open-source relational database management system.
- **Elasticsearch**: Distributed search and analytics engine.

**Other**

- **Django Celery**: Distributed task queue system for Django.
- **Node.js Bull**: A premium queue package for handling background jobs in Node.js.
